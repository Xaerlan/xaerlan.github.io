Agar.io Clone

Main scene of the project

This project was the HTML course, since our professor is passionate about video games as a software development branch, an online game was to be the final semester project.

It was proposed to be developed in Unity, be multiplayer and integrated into web, this allowed me to make something different from the rest of the class, originally, I didn't know how to turn a Unity game into a web game; during research I discovered that the game could be compiled into HMTL5/WebGL.

I already knew how to host a web in Heroku from this same class, so it was originally hosted at Heroku, but made it available in SIMMER.io as well.

This turned out to be a good decision, since Heroku started charging for application hosting and lost the access to my game there, it is still available at SIMMER.io and can be tried out, unfortunately I was unable to make the game multiplayer, but was a good experience nonetheless.
