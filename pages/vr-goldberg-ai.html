VR Rube Goldberg Machine in Unity

Main scene of the project

This project was the final project to be turned in so we could graduate and therefore obtain our engineering title. It was made in Unity while integrating Meta's Oculus Quest VR headset.

This was an extremely taxing project, it involved matching our research with Unity and Meta's versioning for a correct behavior inside the game, many drawbacks occurred during the development where some technologies were deprecated.

The game included a log-in screen, mini-game scenarios, the Goldberg machine and IBM's Watson AI, this, because AI was a required by the university as part of the project; the data was stored in a MySQL database all at the same time running on a Docker container.

Many of the objects used in the game were generated in Blender and then ported into Unity to be used as props.
